## Sikkerhed & Dataisolering

Helix er bygget med sikkerhed som fundament — ikke som et efterfølgende lag:

- **Database-niveau isolation** — Hver organisations data er fysisk adskilt via Row-Level Security
- **Zero-trust arkitektur** — Selv ved softwarefejl kan data ikke lække mellem organisationer
- **Krypteret kommunikation** — Al trafik krypteret via Cloudflare (HTTPS/TLS)
- **Rollebaseret adgangskontrol** — Brugere ser kun det de har adgang til

> Sikkerhed håndhæves i databasen — ikke i applikationskoden. Det betyder at isolation aldrig kan omgås.
